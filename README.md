# 🧮 Calculator App

A simple and modern calculator built using HTML, CSS, and JavaScript.  
This project is created as part of my beginner frontend development practice.

## 🚀 Project Status

- Basic calculator functionality completed
- UI improved with modern dark theme

## ✨ Features

- Basic arithmetic operations (+, -, *, /)
- Clear screen (C button)
- Delete last digit (⌫)
- Decimal support
- Error handling
- Modern UI with hover effects
